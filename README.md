# Unity Gameplay Systems

This repository contains selected C# scripts from a 3D Unity game currently in development.

The full project is not included as it is planned for commercial release.  
These scripts demonstrate core gameplay systems and programming patterns used in the project.

## Included Systems

### Enemy AI
- Finite State Machine behavior
- NavMesh pathfinding
- Line-of-sight detection using raycasting
- Coroutine-based attack logic

### Minion AI
- Finite State Machine controlling worker behavior
- Dynamic mine selection and resource collection
- Navigation using Unity NavMesh
- Delivery system interacting with other gameplay systems
- Idle and repathing logic when paths become blocked

### Stair Construction System
- Grid-based placement system
- Coroutine-driven build animations
- AnimationCurve-driven interpolation
- Dynamic object spawning and transformation

## Technologies
- C#
- Unity Engine
- NavMesh AI
- Coroutines
- Object-Oriented Programming
